Proyek Saya - Coding dan Pengembangan Perangkat Lunak

Aplikasi Manajemen Tugas

Proyek aplikasi manajemen tugas menggunakan React dan Node.js. Aplikasi ini memungkinkan pengguna untuk membuat, mengedit, dan menghapus tugas dengan antarmuka pengguna yang responsif dan intuitif.

  • Teknologi: React, Node.js, MongoDB
